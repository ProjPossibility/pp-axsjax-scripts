1. What is AxsJAX Enhanced Sudoku

AxsJAX Enhanced Sudoku is a regular sudoku game that was AxsJAXed. Sudoku is number puzzle consisting of a 9x9 Grid where each row should contain the numbers from 1 to 9 without duplication. Similarly each column should contain the numbers from 1 to 9 without duplication. Moreover, the 9x9 Grid is divided into 9 3x3 Grids and each such Grid should contain the numbers between 1 to 9 without duplication. For more details, please see the wikipedia article on sudoku.

2. What are the new features

The AxsJAX enhancements provide keyboard navigation for the game. When are on the game page, the following keys are available.

Arrow keys

 When the game page loads, we announce the contents of row 0, column 0 of the grid. After that the arrow keys can be used to navigate the grid. The contents and coordinates of the cell would be announced.

1-9

 Use the keys from 1 to 9 to enter a value into a cell that is currently blank. You have to first navigate to the cell using arrow keys before entering a value into it.

0

 Use 0 to clear the contents of the current cell

r

 reads the contents of the current row that you are on

c

 reads the contents of the current column that you are on

Space

 The space bar can be used to check the current status of the Grid. If there are errors in the Grid, we would announce how many errors are there and the coordinates of each error cell. If there are no errors then we would announce how many cells are remaining to be completed. If the Grid is successfully completed, we would make a announcement to that effect.

e

 to start a new 'easy' game

m

 to start a new 'medium' game

h

 to start a new 'hard' game

v

 to start a new 'evil' (very hard) game

3. How to start using the AxsJAX enhanced Sudoku

All you need to do is make sure that you have the following:

  1. A modern Web browser like Firefox 2.0 or later that supports W3C ARIA.
  2. An adaptive technology that will respond correctly to the accessibility enhancements introduced by W3C ARIA. In particular, many of the enhancements injected by AxsJAX depend on support for live regions a feature that enables adaptive technologies like screen readers and self-voicing browsers deal correctly with asynchronous updates to portions of a Web page.
If your setup meets these requirements, you would need to install GreaseMonkey and then install the GreaseMonkey script for AxsJAX enhancements for sudoku . Restart the browser. Open the sudoku web site and start playing. Please note that while there are numerous sudoku web sites on the internet, the enhancements were developed only for http://view.websudoku.com/ as a proof of concept.

4. Future Direction

We can add error checking when the user is inputting value to a cell to the Grid. I thought it would be too intrusive and left it out. Also, it would be good to combine the assistive and non-assistive navigation techniques so that when the user clicks on a particular cell, the assistive navigation would be updated to be at the clicked cell.
